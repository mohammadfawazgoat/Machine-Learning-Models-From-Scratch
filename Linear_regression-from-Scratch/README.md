# Linear Regression from Scratch

This project implements Linear Regression from scratch using Python and NumPy without using Scikit-learn's LinearRegression model.

The goal of this project was to understand the mathematics behind linear regression, including gradient descent, instead of relying on pre-built machine learning libraries.

## Features

- Implemented Linear Regression from scratch
- Gradient Descent optimization
- Train/Test split for model evaluation
- Prediction on unseen data
- Data visualization using Matplotlib
- Model evaluation using R² Score and Mean Squared Error (MSE)

## Technologies Used

- Python
- NumPy
- Pandas
- Matplotlib
- Scikit-learn (for dataset loading, train-test split, and evaluation metrics)

## Results

- R² Score: 0.8887
- Mean Squared Error (MSE): 56,166,620.77
