# 🤖 Machine Learning Models From Scratch

Welcome to my **Machine Learning Models From Scratch** repository!

This repository contains implementations of popular Machine Learning algorithms built **from scratch using Python and NumPy** to understand how they work internally rather than relying on built-in implementations.

---

# 🎯 Objective

- Learn Machine Learning from first principles.
- Implement algorithms using Python and NumPy.
- Build a strong mathematical foundation.
- Document my learning journey.
